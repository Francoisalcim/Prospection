Memory-safe behavior

  • Raw ClinicalTrials.gov JSON is not stored after each page is processed.
  • Results are deduplicated by NCT ID while streaming.
  • Status is applied via the API. Phase, date, country, and organization are applied locally.
  • ALL mode runs in the background and stores an Excel file instead of keeping the full dataset in memory forever.
