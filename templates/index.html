Important parsing rule

  • Commas split the input first. Each comma-separated item becomes one API query.
  • Uppercase AND/OR inside an item is treated as Boolean logic.
  • Lowercase and/or is treated as normal medical wording.
  • Results are deduplicated by NCT ID.
